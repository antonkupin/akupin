Step 5: Optimize Images

Before uploading, optimize your images:

  • Resize to 400x250px for portfolio previews
  • Use JPG format for photos, PNG for graphics
  • Keep file sizes under 200KB for fast loading
  • Use tools like TinyPNG or ImageOptim

Step 6: Commit and Push

After adding your images, commit them to GitHub:

git add assets/tiypn-photoshoot-*.jpg git commit -m "Add Tiypn photoshoot images" git push origin main
